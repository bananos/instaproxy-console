instaproxy-console
==================
A console tool to work with `InstantProxies service <http://instantproxies.com>`_.

Usage
------------

.. code:: bash

    bananos@amber: $ pip install https://github.com/bananos/instaproxy-console/archive/master.zip


.. code:: bash
